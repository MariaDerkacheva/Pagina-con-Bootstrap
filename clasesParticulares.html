Clases Individuales

Este curso es ideal para aquellas personas que estan interesadas estudiar el idioma de forma constante.
Las clases se dan uno a uno, es decir un estudiante y un profesor. Los encuentros se hacen atravez de la plataforma Zoom,10 min antes de arrancar la clase se manda el link para entrar a la reunion.

Antes de comenzar con nuestras clases se pedira que se haga un test para saber que nivel tiene y cuales son las necesidades del alumno.
Las clases particulares tiene el beneficio de tener una mayor eleccion en los horario disponibles, este sera acordado entre el alumno y el profesor.
Nuestras curso se puede adquirir por paquetes de 2, 6, 10 o mas clases.
